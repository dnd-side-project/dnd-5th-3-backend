## 💻 Tech Stack

- Skills

  > Spring Boot, Spring Security, Data JPA(+Querydsl), Mysql 

- Test

  > JUnit5, Mockito, Jacoco

- Document

  > RestDocs

- Infra

  > Docker Compose

- CI/CD

  > AWS CodeBuild, Github Actions

- ETC

  > JWT, Oauth2(google)
